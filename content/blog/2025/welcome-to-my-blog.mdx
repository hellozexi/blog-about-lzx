## 欢迎阅读我的第一篇博客文章！

大家好！这是我新搭建的个人博客，使用了以下技术栈：

### 技术栈

- **Next.js 15** - 使用最新的 App Router
- **TypeScript** - 类型安全
- **Tailwind CSS** - 实用优先的样式框架
- **shadcn/ui** - 现代化的组件库
- **MDX** - 在 Markdown 中使用 React 组件

### 博客特色

1. **深色模式支持** - 自动适应系统偏好，也可手动切换
2. **文章分类和标签** - 方便组织和查找内容
3. **搜索功能** - 快速找到你想看的文章
4. **评论系统** - 基于 GitHub Discussions，欢迎交流讨论
5. **响应式设计** - 在手机、平板和电脑上都有良好的阅读体验

### 内容规划

我计划在这里分享：

- 技术学习笔记和总结
- 项目开发经验
- 生活感悟和随笔
- 工具和资源推荐

感谢你的阅读，希望你能在这里找到有用的内容！如果有任何问题或建议，欢迎在评论区留言。
