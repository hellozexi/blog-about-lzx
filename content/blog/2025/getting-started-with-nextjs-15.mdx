## Next.js 15 简介

Next.js 15 是一个强大的 React 框架，提供了许多开箱即用的功能，帮助你快速构建高性能的 Web 应用。

### 核心特性

#### 1. App Router

Next.js 15 引入了全新的 App Router，基于 React Server Components 构建。

使用 App Router 可以更方便地组织代码和路由。

#### 2. Server Components 默认启用

服务器组件可以减少客户端 JavaScript 体积，提升性能。

#### 3. 简化的数据获取

使用 async/await 语法获取数据变得更加简单。

### 安装和设置

创建新的 Next.js 项目：

```bash
npx create-next-app@latest my-app
```

运行开发服务器：

```bash
npm run dev
```

### 总结

Next.js 15 提供了强大而灵活的工具，让构建现代 Web 应用变得更加简单。开始你的 Next.js 之旅吧！
